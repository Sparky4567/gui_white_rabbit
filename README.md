# SOME INFORMATION

## ACTIVATION

```

source ./venv/bin/activate

py bot.py or python3 bot.py

```

## DEACTIVATION

```
write exit if you're in typing mode

write in deactivate if you're in typing mode

```

```

ctrl + c to cancel the script while running voice recognition

write in deactivate

```
